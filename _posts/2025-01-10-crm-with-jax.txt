In this project, I developed a multi-producer, multi-injector Capacitance–Resistance Model (CRM) using automatic differentiation in JAX to estimate interwell connectivity and reservoir time constants from historical injection and production data.

CRM is a physics-based reduced-order model widely used in reservoir surveillance and waterflood management. Unlike full-physics simulation, CRM provides fast, interpretable insights into how injectors influence producers — making it highly suitable for real-time field optimization and data-driven reservoir management.

This implementation leverages JAX for differentiable programming, enabling efficient gradient-based calibration of CRM parameters at field scale.

# 2. CRM Mathematical Formulation <a name="crm-mathematical-formulation"></a>

For a system with:

- Np producers  
- Ni injectors  

The discrete-time CRM equation is:

q_p(t+1) = alpha_p * q_p(t)  
           + (1 - alpha_p) * sum( f_p,i * q_inj,i(t) )

Where:

- f_p,i = connectivity between injector i and producer p  
- tau_p = time constant  
- alpha_p = exp( - dt / tau_p )

Matrix form:

q_p(t+1) = alpha ⊙ q_p(t)  
           + (1 - alpha) ⊙ ( F @ q_inj(t) )

Physical constraints:

- f_p,i ≥ 0  
- sum_i f_p,i = 1

# 4. JAX-Based CRM Implementation <a name="jax-based-crm-implementation"></a>

```
import jax
import jax.numpy as jnp

def crm_forward_multi(q_inj, q_p0, F, tau, dt):
    alpha = jnp.exp(-dt / tau)

    def step(q_p, q_inj_t):
        influence = F @ q_inj_t
        q_p_next = alpha * q_p + (1 - alpha) * influence
        return q_p_next, q_p_next

    _, q_p_hist = jax.lax.scan(step, q_p0, q_inj)
    return q_p_hist
```

# 5. Parameter Estimation & Convergence <a name="parameter-estimation--convergence"></a>

```
def loss_fn(params, q_inj, q_p0, q_p_obs, dt):
    F, tau = params
    q_p_pred = crm_forward_multi(q_inj, q_p0, F, tau, dt)
    return jnp.mean((q_p_pred - q_p_obs)**2)

loss_and_grad = jax.value_and_grad(loss_fn)
```

---

# 6. Optimization Loop with Stopping Criteria <a name="optimization-loop-with-stopping-criteria"></a>

```
def optimize_crm(
    q_inj,
    q_p0,
    q_p_obs,
    F_init,
    tau_init,
    dt,
    lr=1e-2,
    max_iter=2000,
    eps_loss=1e-6,
    eps_grad=1e-4
):
    F = F_init
    tau = tau_init
    prev_loss = jnp.inf

    for k in range(max_iter):
        loss, (dF, dtau) = loss_and_grad((F, tau), q_inj, q_p0, q_p_obs, dt)

        grad_norm = jnp.sqrt(jnp.sum(dF**2) + jnp.sum(dtau**2))
        loss_change = jnp.abs(prev_loss - loss)

        F = F - lr * dF
        tau = tau - lr * dtau

        if (loss_change < eps_loss) and (grad_norm < eps_grad):
            print(f"Converged at iteration {k}")
            break

        prev_loss = loss

    return F, tau
```
